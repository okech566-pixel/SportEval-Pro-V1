# SportEval Pro V1

**Professional Windows Application for Physical Education Assessment**

Plateforme intelligente d'évaluation en Education Physique et Sportive (EPS) avec IA intégrée.

## 🎯 Vue d'ensemble

SportEval Pro est une application Windows professionnelle conçue pour les enseignants d'EPS qui permet:
- ✅ Gestion complète des élèves, classes et établissements
- ✅ Évaluation automatisée avec barèmes configurables
- ✅ Analyse prédictive basée sur l'IA
- ✅ Génération de rapports (PDF, Excel, Word, PowerPoint)
- ✅ Statistiques avancées et visualisations
- ✅ Assistant conversationnel pour recommandations

## 🚀 Démarrage Rapide

### Prérequis
- Python 3.11+
- Windows 10/11
- pip

### Installation

```bash
# Cloner le dépôt
git clone https://github.com/okech566-pixel/SportEval-Pro-V1.git
cd SportEval-Pro-V1

# Créer environnement virtuel
python -m venv venv
venv\Scripts\activate

# Installer dépendances
pip install -r requirements.txt

# Lancer l'application
python src/main.py
```
